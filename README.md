# ryazon: another Markov chain text generator

ryazon is a tool for generating text using Markov chains

## example data

example JSON files from the Iliad can be found in the `test_data` directory

## usage

to use Ryazon, simply run:

```bash
ryazon [PATH]
```

where `[PATH]` is the path to a JSON object, array, or string
